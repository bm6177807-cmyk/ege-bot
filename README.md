# 🎓 ЕГЭ Бот — Репетитор ЕГЭ 2026

Telegram-бот для подготовки к ЕГЭ. Задания по 11 предметам, ежедневные тренировки, разбор ошибок, прогресс и мини-пробники.

## Предметы

Русский, Математика, Физика, Химия, Информатика, Биология, История, География, Обществознание, Литература, Английский.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Запустить бота / приветствие |
| `/help` | Список команд |
| `/stats` | Моя статистика |
| `/topics` | Прогресс по темам |
| `/daily` | Ежедневная цель |
| `/remind HH:MM` | Установить напоминание |
| `/remind_off` | Отключить напоминание |
| `/favorites` | Избранные конспекты |
| `/feedback` | Написать администратору |

## Меню предмета (2 уровня)

При выборе предмета открывается:

- **🧩 Практика** — случайное задание, выбор темы, экзамен, фото-задание, тест на уровень, шпаргалки, **ежедневное задание**
- **🧰 Инструменты** — справочники, конвертеры, тренажёры (зависит от предмета)
- **🎯 Пробник (мини)** — 5 вопросов подряд + результат и рекомендации
- **📈 Прогресс** — счётчики решено/верно/неверно, стрик дней, слабые темы
- **❌ Ошибки** — банк ошибок: повторить случайную, список, удалить
- **🏠 Главное меню**

## Новые фичи (v2)

### 🎯 Ежедневное задание (по предметам)
- Одно задание в сутки на каждый предмет
- Кнопки: «Ответить», «Следующее», «Добавить в ошибки»
- Хранится дата выдачи, задание не меняется до следующего дня

### ❌ Ошибки (по предметам)
- «Добавить в ошибки» — из любого задания
- «Повторить ошибки» — случайная из банка
- «Удалить из ошибок»

### 📈 Прогресс (по предметам)
- Решено / Правильных / Неправильных
- Точность в процентах
- Стрик (дней подряд с ежедневным заданием)
- Слабые темы (если решено ≥3 заданий по теме)

### 🎯 Мини-пробник (по предметам)
- 5 вопросов подряд из случайных тем предмета
- Итог: балл + рекомендация
- Время прохождения

### 🧰 Инструменты по предметам

| Предмет | Инструменты |
|---------|-------------|
| Математика | Справочник формул (алгебра, геометрия, тригонометрия, теорвер) |
| Физика | Физические константы + таблицы единиц измерения |
| Информатика | Перевод систем счисления (2/8/10/16) |
| География | Тренажёр «Страна–столица» |
| История | Карточки дат |
| Биология | Решётка Пеннета |
| Английский | Тренажёр слов ЕГЭ (SRS, 30 слов) |
| Химия | Таблица Менделеева + справочник реакций |

## Деплой

### Render
- Build: `pip install -r requirements.txt`
- Start: `python bot.py`
- Переменные окружения: см. ENV.md

### Docker
```bash
docker build -t ege-bot .
docker run -e BOT_TOKEN=... ege-bot
```

## Переменные окружения

Подробное описание — в `ENV.md`.

**Обязательные:** `BOT_TOKEN`, `BOT_USERNAME`, `ADMIN_IDS`  
**YooMoney:** `YOOMONEY_RECEIVER`, `YOOMONEY_WEBHOOK_SECRET`, `PUBLIC_BASE_URL`  
**AI:** `POLZA_AI_API_KEY` (опционально)
